# Tic Tac Toe Game

## Overview
This repository contains a simple implementation of the classic Tic Tac Toe game in Java. 
The game supports two players taking turns to make moves on the 3x3 grid, with the goal of getting three of their symbols in a row.

## Features
- Console-based user interface.
- Two-player mode (Player vs Player).

## Getting Started
1. Clone the repository to your local machine.
   git clone https://github.com/ohyaowen/tic-tac-toe-game.git
2. Compile the Java source file.
   javac TicTacToe.java
3. Run the game
   java TicTacToe
